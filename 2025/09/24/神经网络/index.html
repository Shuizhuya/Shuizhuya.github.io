神经网络

构建神经网络可以学习非线性关系(可以自动学习)。数学表示：y’ = b + w1 x1 + w2 x2 + w3 x3

  • 其中x3是x1和x2之间的特征交叉项。

  • 降低权重(w3)会减少输出值；提高偏差值(b)会增加输出值。

隐藏层：位于输入层和输出层之间的层。每个隐藏层都由一个或多个神经元构成。

神经元：隐藏层中的节点。每个神经元都会执行以下两个步骤：

  1. 计算输入值与其对应权重的乘积之和（即加权总和）。
  2. 将加权求和的结果作为输入传递给激活函数。

隐藏层中每个神经元的值的计算方式与线性模型的输出计算方式相同：将每个输入与其对应的权重参数乘积之和加上偏置值求得。同样，下一层中的神经元则是以隐藏层神经元的值作为输入来计算的。

激活函数

定义：使神经网络能够学习特征和标签之间的非线性(复杂)关系的函数。

添加了神经网络隐藏层后运算仍然是线性的，不能表示非线性关系，所以需要在输出每个节点的值之前使用激活函数。

常见的激活函数

三个常用的激活函数：sigmoid、tanh和ReLU。

  • sigmoid(S型)函数对输入执行以下变换x，产生介于0到1之间的输出值：F(x) = 1 / (1 + e-x)

  • tanh(双曲正切)函数将输入x产生介于 -1和1 之间的输出值：F(x) = tanh(x)

    tanh(x) = (ex - e-x) / (ex + e-x)或tanh(x) = 2 * sigmoid(2x) - 1

  • ReLU(修正线性单元)函数使用一下算法转换输出：

    • 如果x小于0，返回0；

    • 如果x大于或等于0，返回输入值。

    用数学形式表示：F(x) = max(0, x)

ReLU 作为激活函数通常比 sigmoid 或 tanh 等平滑效果更好，且它在神经网络训练过程中不易受到梯度消失问题的影响，计算也更容易。

反向传播

定义：​根据模型预测的损失，从输出层开始，反向逐层计算神经网络中每个权重参数对于该误差的梯度，从而指导如何更新这些权重以最小化误差。神经网络中实现梯度下降的算法。

梯度消失

较低神经网络层（靠近输入层）的梯度可能非常小。当较低层的梯度值接近于 0 时，梯度就被称为“消失”。梯度消失的层训练速度非常慢，甚至根本不训练。

ReLU 激活函数可以帮助防止梯度消失。

梯度爆炸

如果网络中的权重非常大，那么较低层的梯度会包含许多大项的乘积。在这种情况下，可能会出现梯度爆炸：梯度过大而无法收敛。

批量标准化可以帮助防止梯度爆炸，也可以降低学习率。

死ReLU单位

若 ReLU 单元的加权和低于 0，该 ReLU 单元就会陷入困境。它的输出为 0，对网络的输出没有任何贡献，并且在反向传播过程中梯度也无法再流经它。

降低学习率有助于防止 ReLU 单元消亡。

Dropout正则化

定义：一种正则化形式，称为丢弃正则化。丢弃正则化: 一种用于训练神经网络的正则化形式

随机地在网络中“丢弃”单个梯度步骤中的单元激活值。丢弃的单元越多，正则化效果越强：

  • 0.0 = 无 dropout 正则化。
  • 1.0 = 删除所有节点。模型不会学习任何内容。
  • 0.0 到 1.0 之间的值 = 更有用。

多类别分类

一对多

一对多提供了一种使用二元分类对多个可能标签进行一系列是或否预测的方法。

但当类别总数增多时，效率会变得越来越低。使用神经网络创建一个“一对多”模型能显著提升效率。

一对一

在一对多方法中，每个二元结果集的概率(输出层使用S型激活函数，每个输出值代表输入为该类别的概率)都是独立于其他集合的，所以加起来的总值不一定等于1.0。而一对一方法算出来的概率总和等于1.0。可以用到softmax函数(放在输出层之前的隐藏层，须具有与输出层相同的节点数)，它为多分类问题中的每个类别分配十进制概率。

softmax选择

softmax变体：

  • 完全softmax：一般情况下的softmax，即为每个可能的类别计算一个概率。

  • 候选采样：softmax会计算所有正类别的概率，和随机样本的负类别概率。

类别数量较少时可以使用完全softmax；类别数量增多，计算成本会变高，使用候选采样可以提升效率。
