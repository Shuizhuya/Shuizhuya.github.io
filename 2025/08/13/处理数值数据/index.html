处理数值数据

数值数据：用实数或整数表示的特征。

模型提取数据的方式并不是直接对数据集的行执行操作，而是会提取一个浮点值数组，称为特征向量。

  • 特征向量：构成示例的特征值数组。在训练和推理期间作为输入。

特征向量通常不会直接使用数据集的原始值。相反的，需要将数据集的值进行处理，转化为模型能够更好地学习的表示形式。

特征工程

定义：将原始数据集值表示为特征处理中可训练值的最佳方法。有时也被称为特征提取或特征化。包括以下步骤：

  1. 确定哪些特征可能在训练模型方面非常有用；

  2. 将数据集中的原始数据转换为这些高效特征的版本。

常见的特征工程技术有：

  • 归一化：将数值转换为标准范围。

  • 分桶(分箱)：将数值转换为范围分桶。

初始步骤

1.将数据可视化

图表可以帮助我们发现数据中隐藏的异常或模式，所以，在深入分析之前，需要先以图形的形式(散点图/直方图)查看数据。建议用pandas库。

2.对数据进行统计评估

通过数学方法评估潜在的特征和标签，收集以下基本统计信息：

  • 平均值和中位数
  • 标准差
  • 四分位数分界点的值：第 0、25、50、75 和 100 百分位。第 0 个百分位数是此列的最小值；第 100 个百分位数是此列的最大值。（第 50 百分位是指中位数。）

比较标准差与平均值也可以看出是否有异常：

  1. 均值和标准差都小，说明数据的值比较稳定，波动性较小。

  2. 均值大，标准差小，说明数据的值比较稳定，没有太大波动。

  3. 均值小，标准差大，说明数据的值比较分散，波动较大。

  4. 均值大，标准差大，说明数据的值分散，波动大。

3.查找离群值

  • 离群值：是指与特征或标签中的大多数其他值相差甚远的值。

离群值常常会导致模型训练出现问题，因此需要找出它。

如果第 0 百分位数与第 25 百分位数之间的差异与第 75 百分位数与第 100 百分位数之间的差异明显不同，则数据集可能包含离群值。

标准化

定义：将变量的实际值范围转换为标准值范围的过程，也叫归一化。

注意：如果在训练期间对某项特征进行归一化处理，那么在进行预测时也必须对该特征进行归一化处理。

常用的归一化方法

1.线性缩放

定义：指将浮点值从其自然范围转换为标准范围。通常使用减法和除法的方式将原始值替换为介于-1到+1或0到1之间的数字。

数学公式(缩放到0到1范围)：x’ = (x - xmin) / (xmax - xmin)

  • x’是缩放后的值。

  • x是原始值。

  • xmin是相应特征的数据集中的最低值。

  • xmax是相应特征的数据集中的最高值。

适用条件(扁平状)

1. 数据的下限和上限不会随时间发生太大变化。

2. 特征包含的离群值很少或没有，并且这些离群值并不极端。

3. 该特征在其范围内大致呈均匀分布。 也就是说，直方图会显示大致均匀的条形，对应于大多数值。

2.Z分数缩放

定义：Z得分是指某个值与平均值相差的标准差数量(高为+，低为-)。

数学公式：x’ = (x - μ) / σ

  • x’是Z得分。

  • x是原始值。

  • μ是平均值。

  • σ是标准差。

适用条件(钟形)

数据遵循正态分布或某种程度上类似于正态分布。

3.对数缩放

定义：计算原始值的对数。理论上，对数可以是任意底数；但实际中，对数缩放通常会计算自然对数(ln)。

数学公式：x’ = lnx

  • x’是x的自然对数。

  • x是原始值。

适用条件(重尾形)

数据符合幂律分布。幂律分布：

①x的值较低时，y的值非常高；②随着x值得增加，y值会快速减小，即，x的值较高时，y的值会非常低。

4.裁剪

定义：一种处理离群值的方法。通常通过执行以下一种或两种操作来实现：

  • 将大于最大阈值的特征值减小到最大阈值。

  • 将小于最小阈值的特征值增加到最小阈值。

在归一化前/后裁剪都可以。

分箱

定义：是一种特征工程，将不同的数值子范围分组到分箱或存储分区(也称为分桶)。

适用条件

当以下任一条件成立时，分箱是一种优于缩放或裁剪的替代方法：

  1. 该特征与标签之间的整体线性关系较弱或者根本不存在。

  2. 当特征值被聚类时。

只有当某个容器中有足够多的样本时，模型才能学习该容器与标签之间的关联。

分位数分桶

定义：可创建分桶边界，使每个分桶中的样本数完全匹配或几乎相等。分位数分桶大部分情况下隐藏离群值。

避免了使用普通分桶时，导致一些桶的示例数量足够进行训练，而有些桶示例数量很少，不能进行模型训练。

多项式转换

定义： 一种特征，不在输入特征之列，而是从一个或多个输入特征组装而来，也叫合成特征。创建方法包括：

  • 对连续特征进行分桶。

  • 创建特征组合。

    特征组合： 通过“组合”分类或分桶特征形成的合成特征。

  • 将一个特征值与其他特征值或其本身相乘（或相除）。

  • 对特征值应用超越函数。例如，如果 c 是一个输入特征，那么以下就是合成特征的示例：

    • sin(c)

    • ln(c)
