HTML基础

定义
HTML是超文本标记语言，不同于编程语言；使用标签来描述网页

    由一对尖括号(“<>”)包住的元素叫做标签。通常情况下一个完整的标签由一个开始标签和结束标签组成，结束标签需在元素前加斜杠”/“，用于表示标签的结束。

    写在开始标签和结束标签内的叫内容，即这个标签需要实现的内容。

    只有开始标签，没有结束标签的标签叫单标签。

结构

    用于表明当前文件是一个HTML5文件，避免网页解析错误。

    整个文档的最顶层标签/根标签，是所有标签的父标签。

    head标签内的内容，不会直接在页面中展示，用于定义文档的属性。

    body标签内的内容则是需要展示在页面中的内容。

meta元素

用法

    1.申明字符编码：<meta charset = "UTF-8">，避免解码错误。

    2.页面尺寸自适应：<meta name="viewport" content="width=device-width, initial-scale=1.0">

    3.给网页优化搜索引擎：<meta name=" " content=" ">

name=”keywords” ，添加网页的关键字/词。

name=”description”，描述网页的内容，在content中写入与网页相符的简述。

name=”author”，网页的作者。

    4.自动跳转到链接：<meta http-equiv="refresh" content="3; url">，content中填写时间(几秒后跳转)和网页链接。

无结束标签。

style元素

作用：为文档定义样式信息
用法

    1.改变文本样式：margin-left/right/top: 10%/px/em分别是左/右/顶外边距

    2.media：给指定设备的设置。例打印机样式：<style media="print"></style>；除此之外，还能用于在特定尺寸中显示样式：<style media="screen and (min-width: 512px) and (max-width: 1024px)">，min指不低于，max指不大于，即在512-1024px中显示。

link元素

作用：链接外部样式表
用法

rel属性是必须添加的，它决定了浏览器对link元素的方式，即link和当前文档的关系。

    1.链接外部样式表：<link rel="stylesheet" type="text/css" href="url">，stylesheet表示链接的是样式表。

    2.media属性：和style一样，media属性表示用于指定链接显示在什么设备上。但很多设备在HTML5中已经弃用(剩all，print，screen)。

    3.链接网站图标：<link rel="icon" type="image/x-icon" href="url">。

无结束标签。

base元素

作用：用于设置文档的基准url，让文档中的相对链接在此基础上进行解析
用法

    

    2.载入外部脚本：<script type="text/javascript" src="url">。

另外，还可实现延迟执行脚本

    由于HTML的解析是单线操作，所以在解析时碰到脚本会暂停HTML的解析去加载脚本并执行，但有时，我们需要先完成HTML的解析，再执行脚本，这就叫延迟执行脚本。例：在HTML解析完成之后把所有段落的内容都改为”xxx”。

    添加defer属性可以实现。但只能用于外部脚本文件，对内嵌脚本不起作用。

异步执行脚本

    避免脚本加载时间过长影响HTML文档的解析。

    添加async属性可以异步执行脚本，即脚本加载完之后自动执行。异步执行脚本会使脚本无序化，先加载完的就先执行。同样只适用于外部脚本，对内嵌脚本无效。

注释

①HTML：格式为<!-- xxx -->。

②CSS：格式为/* xxx */。

③JavaScript：单行注释// xxx，多行/* xxx */。

块级元素和行内元素

块级元素总是在新的行上开始，并尽可能地占本行全部可用的宽度。

    所有块级元素：<address> <article> <aside> <blockquote> <canvas> <dd> <div> <dl> <dt> <fieldset> <figcaption> <figure> <footer> <form> <h1>-<h6> <header> <hr> <li> <main> <nav> <noscript> <ol> <output> <p> <pre> <section> <table> <tfoot> <ul> <video>

行内元素不会另起一行，它只占用必要的宽度。

    所有行内元素：<a> <abbr> <acronym> <b> <bdo> <big> <br> <button> <cite> <code> <dfn> <em> <img> <input> <kbd> <label> <map> <object> <q> <samp> <script> <select> <small> <span> <strong> <sub> <sup> <textarea> <time> <tt> <var>

区别：

一般情况下，行内元素只能包含数据和其他行内元素；

而块级元素可以包含行内元素和其他块级元素；

这种结构上的包含继承区别可以使块级元素创建比行内元素更“大型”的结构。

span元素

作用：与p元素对应的行内元素
用法

    可以用来表示不需要换行的文本，或者在p的内容里增加span，可以给span元素包住的内容添加样式等。

br元素

作用：在文档中表示换行符
用法

    在文档中插入进行换行，也可实现行内元素独占一行的效果。

无结束标签。

pre元素

作用：定义预格式化文本。预格式化即保留文本在源代码中的格式，使得页面中显示的和源代码中的效果完全一致。

HTML字符实体

作用：用于在文档中正确显示预留字符。避免浏览器当成关键字错误解析。
用法

    显示空格/代码。字符实体查询参考HTML 字符实体 | 菜鸟教程。

    字符实体最后都以”;”结尾。

code元素

作用：定义计算机代码片段
用法

    可以和pre元素一起实现语义化，单独用code元素会取消代码的缩进。

var元素

作用：定义程序的变量

kbd元素

作用：定义用户的输入

samp元素

作用：定义程序的输出

引用元素

1.q元素

作用：定义比较短的引用。浏览器在引用的内容两侧添加引号。
用法

    给内容加上双引号。

2.blockquote元素

作用：用于长文本的引用。浏览器使用缩进形式来显示文本。
用法

    用cite属性指定引用文本来源，<blockquote cite="url"> </blockquote>。

q和blockquote都用于实现HTML的语义化，而不是呈现表示，呈现的样式由CSS实现。

3.cite元素

作用：定义作品的标题。浏览器用斜体表示。
用法

    给内容标记成标题。

4.abbr元素

作用：定义简称/缩写。浏览器用点划线表示。
用法

    配合title属性可以指定该缩写代表的完整含义，<abbr title="缩写的完整含义/解释"> </abbr>

5.dfn元素

作用：表示定义的术语。浏览器用斜体表示。
用法

    同样可以配合title属性指明术语的值；

    若没有title属性，但内容中有abbr标签，且它有title属性，则会直接用abbr中title的值表示术语的值。

6.address元素

作用：定义文档或者文章的作者/拥有者的联系信息/方式。浏览器用斜体表示。
用法

    可以与a标签搭配使用，实现超链接。

7.ruby，rt，rp元素

作用：三者结合用于给文字注音/字符注释
用法

    ruby元素用于展示被注音的文字/符号；rt元素用来标记注音符号；rp元素用于标记当浏览器不支持ruby元素时显示的内容

8.bdo元素

作用：用于将文本以不同方向显示
用法

    bdo元素中有dir属性，用于指定文本绘制方向：

ltr：从左向右绘制文本；

rtl：从右向左绘制文本。

格式化元素

strong/b元素

作用：strong元素用于强调文本的重要性。两者在浏览器都以粗体表示。
用法

    将需加重/加粗的文本用strong/b标签括起来。

需要注意的是b元素也可以加粗文本，但不具有重要的语义。

em/i元素

作用：em元素用于强调文本。两者在浏览器都以斜体表示。
用法

    将需强调/倾斜的文本用em/i标签括起来。

需要注意的是i元素也可以使文本倾斜，但不具有强调的语义。

del元素

作用：定义一些被删除的内容。浏览器用删除线表示。
用法

    通常和ins标签搭配用，表示删除后的新版。

ins元素

作用：定义被插入的文本。浏览器用下划线表示。
用法

    可以和del标签搭配，可以表示删除后用新插入的文本替代。

del和ins元素都有cite属性，用于指向一个文档的url，这个文档解释了这段文本被删除/插入的原因，一般cite属性在浏览器中不可直接观察到，但是通过屏幕阅读器使用。

datetime属性也是两者都有的，用于规定删除/插入文本的日期和时间，例：<del datetime="2025-06-22T21:52:00Z"></del>同样也是在浏览器中无视觉效果，但能被屏幕阅读器使用。

s元素

作用：定义不正确的文本。浏览器用删除线表示。
用法

    与del元素不同，s元素用于表示错误的语义。

u元素

作用：定义拼写错误的单词或汉语中的专有名词。浏览器用下划线表示。
用法

    与ins元素不同，u元素用于表示拼写错误的单词/汉语中的专有名词。

mark元素

作用：定义被标记的内容。浏览器用高亮表示。
用法

    在需要突出显示文本时使用。

sup/sub元素

作用：sup定义上标文本；sub定义下标文本。
用法

    多用于表示数学公式/化学方程式/上标字母/序数，不可因为仅外观/呈现方面的目的使用。

small元素

作用：定义旁注和小字印刷的文本。浏览器用小一号字体表示。

用法

    可以用来声明版权或法律文本等，不可因为外观/呈现方面的目的使用。

table元素

作用：用于定义一个表格

tr元素

作用：tr = table row，用于定义表格中的行
