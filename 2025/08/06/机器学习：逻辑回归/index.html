机器学习:逻辑回归

逻辑回归

定义：一种可预测概率的回归模型。一般具有以下特征：

  • 标签为分类。逻辑回归通常是指二元逻辑回归，即计算具有两个可能值的标签的概率模型。另一种不太常见的变体是多项式逻辑回归，用于计算有两个以上可能值的标签概率。

  • 训练期间的损失函数为对数损失函数。(具有两个以上可能值的标签，可并行放置多个Log Loss单位)

  • 采用的是线性架构，不是深度神经网络。

S型函数

定义：一种数学函数，可将输入值“压”到受限范围内，通常为0到1或-1到+1。

标准逻辑函数也称为S型函数。 公式为：f(x) = 1/ (1 + e-x)

使用S型函数转换线性输出

逻辑回归模型的线性组件：z = b + ω1x1 + w2x2 + …

  • z是线性方程的输出(也叫对数几率)。

  • b是偏差。

  • ω的值是模型学习的权重。

  • x的值是特定样本的特征值。

要获得逻辑回归预测结果，需要将z值传递给S型函数，得到一个介于0到1之间(不含0和1)的值：y’ = 1/(1 + e-z)。

损失和正则化

逻辑回归模型的训练过程与线性回归的两个关键区别：

  • 逻辑回归模型使用 对数损失函数作为损失函数 而不是平方损失函数。

  • 应用正则化是防止过拟合。

对数损失函数

逻辑回归模型的变化率不是常量。当对数几率(z)值接近0时，与z较大时相比，z的增加会导致y的变化幅度更大(不论正数负数)。

通过对数损失方程式会返回变化幅度的对数，而不只是从数据到预测的距离。

对数损失的计算公式为：LogLoss = Σ(x,y)∈D - y log(y’) - (1 - y) log(1 - y’)

  • (x,y)∈D是包含多个有标签样本的数据集，这些样本为(x,y)对。

  • y是有标签样本中的标签。由于是逻辑回归，y的值必须为0或1。

  • y’是模型的预测结果，同样值也是在0到1之间，给定集合x中的功能。

正则化

定义：降低模型的复杂度，在逻辑研究中回归模型。即任何可减少过拟合的机制。

常见的降低模型复杂度方法：

  • L2正则化。

    L2正则化：一种常用的正则化指标，使用公式：L2 regularization = ω12 + ω22 + … + ωn2 (ω是权重)

    L2正则化会使权重趋近0，但绝对不会让权重完全为0。

  • 早停法。

    早停法：一种不涉及复杂性计算的正则化方法。提前停止训练只是指在模型完全收敛之前结束训练。例如，当验证集的损失曲线开始增加（斜率变为正值）时，您可以结束训练。

    早停法通常会增加训练损失，但可以降低测试损失。
